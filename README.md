# FinSight Dashboard

A recruiter-focused **senior-level frontend dashboard project** built with **Next.js, TypeScript, Tailwind CSS, and shadcn/ui**.

## Features
- Premium dark SaaS UI
- Reusable component architecture
- Custom hooks
- Optimized rendering with React.memo
- Type-safe components
- Responsive design
- Dashboard analytics UI
- Transaction management UI

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

## Run Project
```bash
npm install
npm run dev
